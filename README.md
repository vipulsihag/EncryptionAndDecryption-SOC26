# Encryption and Decryption SOC26
This is the project repository for Summer of Code 2026 by Web and Coding Club, IIT Bombay. The aim of this project is to use various Machine Learning Algorithms and techniques to create models which can decrypt various kind of encrypted messages.
This repo contains my weekly assignments.

## Setup
All of the python code is in jupyter noteboks, so any suitable notebook environment with python 3.10+ would work well. There are external libraries used, so these must also be installed : 
- numpy
- pandas
- matlplotlib
- kagglehub
- torch
- transformers
